## ツールの概要

**Forter ROI Calculator** は、Forter導入によるビジネスインパクトを試算するためのツールです。

### 主な機能

- 📊 **2つのシナリオを比較**: ベースシナリオと比較シナリオを設定し、ROIを計算
- 💾 **自動保存**: 入力内容は自動的に保存されます（手動保存ボタンも利用可能）
- 📄 **エクスポート**: 計算結果をPDFまたはHTML形式でエクスポート
- ⭐ **お気に入り**: よく使うセッションをお気に入りに登録
- 📋 **複製機能**: 既存のセッションを複製して新しい試算を作成
- 👥 **共有**: 複数のユーザーが同じセッションを閲覧・編集可能

## ログイン方法

### 初回ログイン

1. ブラウザでForter ROI Calculatorにアクセス
2. **「新規登録はこちら」** リンクをクリック
3. メールアドレスとパスワードを入力
4. **「登録」** ボタンをクリック
5. 登録が完了すると、自動的にダッシュボードに移動します

### 2回目以降のログイン

1. メールアドレスとパスワードを入力
2. **「ログイン」** ボタンをクリック

### ログイン時の注意事項

- パスワードを5回間違えると、30分間ログインがブロックされます
- ログインに失敗した場合、残り試行回数が表示されます

## ダッシュボードの使い方

### ダッシュボードの構成

ダッシュボードには以下の要素があります：

- **ヘッダー**: 右上にログアウトボタン
- **新規作成ボタン**: 新しいROI計算セッションを作成
- **セッション一覧**: 過去に作成したセッションが表示されます
- **お気に入りフィルター**: お気に入りに登録したセッションのみを表示

### セッションの作成

1. **「+ 新規作成」** ボタンをクリック
2. モーダルが表示されます
3. **企業名**（任意）と**シナリオ名**を入力
4. **「作成」** ボタンをクリック
5. ROI計算画面に移動します

### セッションの操作

各セッションカードで以下の操作が可能です：

- **クリック**: セッションを開く
- **お気に入りボタン（⭐）**: お気に入りに登録/解除
- **複製ボタン**: セッションを複製して新しい試算を作成
- **削除ボタン**: セッションを削除（確認ダイアログが表示されます）

### セッション情報

各セッションカードには以下が表示されます：

- **企業名**: セッション作成時に設定した企業名
- **シナリオ名**: セッションのタイトル
- **作成者**: セッションを作成したユーザーのメールアドレス
- **更新日時**: 最後に更新された日時

## ROI計算の実行

### 計算画面の構成

計算画面は以下のセクションで構成されています：

1. **ヘッダー**: 企業名、シナリオ名、保存ボタン、お気に入りボタン、エクスポートボタン
2. **入力フォーム**: 計算に必要な数値を入力
3. **計算結果**: ファネル、取扱高、コスト、ROIサマリー

### 計算の手順

1. **入力値を設定**: 各入力フィールドに数値を入力
2. **「計算」ボタンをクリック**: 計算結果が更新されます
3. **結果を確認**: ファネル、取扱高、コスト、ROIを確認

### 入力値の保存

- **手動保存**: ヘッダーの**「保存」**ボタンをクリック
- 保存が完了すると、保存ステータスが表示されます

## 入力項目の説明

### 共通の事実（両シナリオに適用）

これらの値は、ベースシナリオと比較シナリオの両方に適用されます。

| 項目 | 説明 | 単位 |
|------|------|------|
| **年間取引試行回数** | 1年間の取引試行回数 | 回 |
| **平均取引額-成功決済** | 成功した決済の平均取引額 | 円 |
| **平均取引額-失敗決済** | 失敗した決済の平均取引額 | 円 |
| **取引毎の3DSコスト** | 3DS認証にかかる1取引あたりのコスト | 円 |
| **取引毎のコスト** | その他の1取引あたりのコスト | 円 |
| **再試行率** | 別カードや別の支払い方法でリトライする割合 | % |

### シナリオ別の入力項目

各シナリオ（ベース/比較）で以下の項目を設定します：

| 項目 | 説明 | 単位 |
|------|------|------|
| **決済前の拒否率** | 決済前（Pre-Auth）に拒否される割合 | % |
| **決済後の拒否率（自動）** | 決済後に自動で拒否される割合 | % |
| **決済後の拒否率（目検）** | 決済後に目視検証で拒否される割合 | % |
| **3DS利用率** | 3DS認証を使用する取引の割合 | % |
| **3DSエラー率** | 3DS認証でエラーが発生する割合 | % |
| **与信成功率** | 与信が成功する割合 | % |
| **チャージバック率** | チャージバックが発生する割合 | % |
| **固定費用** | 年間の固定費用 | 円 |
| **補填費用** | 取引額に対する補填費用の割合 | % |
| **年間目検費用** | 年間の目視検証にかかる費用 | 円 |
| **3DS無しにのみ適用** | チェックを入れると、3DS無しの取引にのみ適用 | - |

### 入力のヒント

- **数値の入力**: カンマ区切りで入力できます（例: `1,000,000`）
- **コメント機能**: 各入力項目にコメントを追加できます（💬アイコンをクリック）
- **非表示機能**: 使用しない項目は「非表示」にチェックを入れると、表示を隠せます
- **再表示**: 「非表示の項目をすべて表示」ボタンで、隠した項目を再表示できます

## 計算結果の見方

### 1. ファネル表示

取引の流れを視覚的に表示します：

- **支払い試行**: 年間取引試行回数
- **決済前の拒否**: Pre-Authで拒否された取引数
- **決済試行**: Pre-Authを通過した取引数
- **3DS失敗**: 3DS認証で失敗した取引数
- **与信数**: 3DSを通過した取引数
- **与信失敗**: 与信で失敗した取引数
- **与信成功数**: 与信が成功した取引数
- **決済後の拒否**: Post-Authで拒否された取引数
- **支払い成功数**: 最終的に成功した取引数

### 2. 取扱高の変化

- **取扱高**: 成功した取引の合計金額
- **リトライを考慮した保守的な取扱高**: 再試行を考慮した保守的な見積もり

### 3. コストの変化

各コスト項目の変化を表示：

- **チャージバックコスト**: チャージバックにかかる費用
- **3DSコスト**: 3DS認証にかかる費用
- **目検コスト**: 目視検証にかかる費用
- **ベンダーコスト**: Forterなどのベンダー費用
- **総費用**: すべてのコストの合計

### 4. ROIサマリー

- **保守的な取扱高増加**: 比較シナリオでの取扱高増加額
- **コスト変動**: コストの増減額
- **投資収益率（ROI）**: 投資に対する収益率

## エクスポート機能

### PDFエクスポート

1. ヘッダーの**「PDFにエクスポート」**ボタンをクリック
2. PDFファイルがダウンロードされます
3. PDFには計算結果と入力値が含まれます

### HTMLエクスポート

1. ヘッダーの**「HTMLをエクスポート」**ボタンをクリック
2. HTMLファイルがダウンロードされます
3. HTMLファイルはブラウザで開き、インタラクティブに操作できます
4. エクスポートされたHTMLは、計算機能、スライダー、非表示機能などが動作します

### エクスポートの注意事項

- PDF/HTMLには、入力値、計算結果、コメントが含まれます
- エクスポートされたファイルは、他のユーザーと共有できます
- HTMLファイルは、インターネット接続なしでも動作します（ただし、初回読み込み時はCDNからCSSを読み込みます）

## その他の機能

### お気に入り機能

よく使うセッションをお気に入りに登録できます：

1. ダッシュボードでセッションカードの**お気に入りボタン（⭐）**をクリック
2. または、計算画面のヘッダーにある**お気に入りボタン**をクリック
3. お気に入りに登録されたセッションは、ダッシュボードでフィルターできます

### セッションの複製

既存のセッションを複製して、新しい試算を作成できます：

1. ダッシュボードでセッションカードの**「複製」**ボタンをクリック
2. または、計算画面のヘッダーにある**「複製」**ボタンをクリック
3. 確認ダイアログで**「はい」**を選択
4. 新しいセッションが作成され、自動的に開きます

### セッションの削除

不要なセッションを削除できます：

1. ダッシュボードでセッションカードの**「削除」**ボタンをクリック
2. 確認ダイアログで**「削除」**を選択
3. セッションが削除されます（この操作は取り消せません）

### ヘッダーの固定表示

計算画面のヘッダーは、スクロールしても常に表示されます。これにより、保存ボタンやエクスポートボタンに常にアクセスできます。

## よくある質問

### Q1. 入力値を保存するには？

**A:** ヘッダーの**「保存」**ボタンをクリックしてください。保存が完了すると、保存ステータスが表示されます。

### Q2. 計算結果が更新されない

**A:** **「計算」**ボタンをクリックしてください。入力値を変更しただけでは、計算結果は自動更新されません。

### Q3. エクスポートしたHTMLが動作しない

**A:** エクスポートされたHTMLファイルは、ローカルファイルシステムから直接開いても動作します。ただし、一部のブラウザではセキュリティ制限により、ローカルファイルからのJavaScript実行が制限される場合があります。その場合は、Webサーバー経由でアクセスするか、PDFエクスポートをご利用ください。

### Q4. セッションを他のユーザーと共有できますか？

**A:** 現在、すべてのユーザーがすべてのセッションを閲覧・編集できます。将来的には、特定のユーザーとの共有機能を追加予定です。

### Q5. パスワードを忘れた場合

**A:** 現在、パスワードリセット機能は実装されていません。管理者に連絡して、パスワードのリセットを依頼してください。

### Q6. ログインがブロックされた

**A:** パスワードを5回間違えると、30分間ログインがブロックされます。ブロック解除時刻が表示されますので、その時刻までお待ちください。

### Q7. 入力項目の意味がわからない

**A:** 各入力項目には説明が表示されています。また、コメント機能を使用して、メモを追加することもできます。

### Q8. 計算結果の数値がおかしい

**A:** 以下を確認してください：
- 入力値が正しく入力されているか
- 単位が正しいか（%と円を間違えていないか）
- 「計算」ボタンをクリックしたか

### Q9. セッションが表示されない

**A:** 以下を確認してください：
- お気に入りフィルターが有効になっていないか
- 別のユーザーでログインしていないか
- ブラウザをリロードしてください

### Q10. エクスポート機能が動作しない

**A:** 以下を確認してください：
- ブラウザのポップアップブロッカーが有効になっていないか
- ダウンロードフォルダの権限を確認
- 別のブラウザで試してください
